Semplicissimo timer in c da eseguire su macchina UNIX. 

**Istruzioni installazione**:

1 - Cloneare la repository: 
```console
git clone https://github.com/roccolr/simple_beeping_timer.git
```

2 - Ottenere l'eseguibile:

```console
make timer 
```

nella directory dove è stata clonata la repository;

**Istruzioni esecuzione**: 

```console
/timer ore minuti secondi  
```


nella directory dove è stata clonata la repository.
